# Education App Wireframe

## 📖 Overview
This project is a low-fidelity wireframe for an Education Mobile Application, created in Figma as part of my CodeAlpha Internship task. The purpose of this project is to design the basic user flow and application layout before moving to the UI design and development phase.

## 🎯 Project Objectives
- Design a simple and user-friendly education app.
- Create a clear navigation flow.
- Plan the app structure before development.
- Improve user experience through wireframing.

## 📱 Screens Included
- Welcome Screen
- Courses Screen
- Course Details Screen
- Login Screen
- Profile Screen

## 🛠️ Tool Used
- Figma

## ✨ Features
- Simple and clean mobile wireframe
- Easy navigation
- Course browsing interface
- Course details page
- User login page
- User profile page
